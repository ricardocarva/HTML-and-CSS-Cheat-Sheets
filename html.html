HTML Table


Common HTML Elements

Tag Name Description
<!DOCTYPE html> DocType The Doctype Declaration must be the very first line of code in an HTML.
<html lang="en"> HTML The tag is the container for all other HTML elements (except for the DOCTYPE tag). Add lang to assist search engines and browsers
<head> Head The tag is a container for metadata placed between the html and body tags.
<link rel="" href=""> Link The tag links the document and an external resource like a CSS file.
<title> Title The Title defines the title of the document. It is shown in the browser's title bar or in the page's tab.
<body> Body The body tag defines the document's body.
<footer> Footer The tag defines a footer for a document or section.
<nav> Navigation The tag defines a set of navigation links.
<header> Header The container for introductory content or set of navigational links
<article> Article The tag specifies independent, self-contained content.
<section> Section The tag defines a section in a document. Check below for more details.
<h1>-<h6> Header 1 - Header 6 The tags for different sizes of Headers. For Examples, see MDN.
<p> Paragraph The tag defines a paragraph.
<a> Anchor The tag defines a hyperlink, which is used to link from one page to another.
<table> Table The tag defines an HTML table.
<thead> Table Head The tag is used to group header content in an HTML table.
<tr> Table Row The tag defines a row in an HTML table.
<th> Table Header The tag defines a header cell in an HTML table.
<tbody> Table Body The tag is used to group the body content in an HTML table.
<td> Table Data The tag defines a standard data cell in an HTML table.
<image> Image The tag is used to embed an image in an HTML page.
<figcaption> Figure Caption The tag defines a caption for a figure element like an Image
<audio> Audio The tag is used to embed sound content in a document, such as music or other audio streams.
<video> Video The tag is used to embed video content in a document, such as a movie clip or other video streams.
<input> Input The tag specifies an input field where the user can enter data. It is the most important form element. Check more here.
<br> Break The tag inserts a single line break.
<div> Division The tag defines a division or a section in an HTML document.
<ul> Unordered list The tag defines an unordered (bulleted) list.
<ol> Ordered list The tag defines an ordered list. An ordered list can be numerical or alphabetical.
<li> List The tag is used to define each list item.
<textarea> Text Area The tag defines a multi-line text input control.
<button> Button The tag defines a clickable button.


Section vs Article

  • Section:

    • This tag is used to split a page into sections like Introduction, Contact Information, Details, etc and each of these sections can be in a different <section> tag. The <section> tag is introduced to wrap-up the things in a particular section. The section tag divides the content into sections and subsections. The section tag is used when requirements of two headers or footers or any other section of documents needed. Section tag grouped the generic block of related content.
  • Article:

    • This tag contains independent content that doesn’t require any other context. So the article tag can be placed inside the main content. But each of the articles will contain independent content within it. Like YouTube use to contain different kinds of videos on a single page, each video is independent or you can see the course page of GeeksforGeeks each course is independent, each course can have its own page.

    Note: Placing <article> tag inside of <section> tag is good practice, like section basically defines the types and the articles will contain the specific contents in that type of section.
